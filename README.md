#  Gerenciador de Tarefas

Aplicação web moderna para gerenciamento de tarefas desenvolvida com **Next.js**, focada em produtividade, organização e experiência de usuário.

 Website: https://gerenciador-de-tarefas-one-pearl.vercel.app/  
 Repositório: https://github.com/murilotecoteco/Gerenciador-de-Tarefas

##  Funcionalidades

- Criar, editar e remover tarefas  
- Marcar tarefas como concluídas  
- Filtros por status (todas, pendentes, concluídas)  
- Sistema de estatísticas de tarefas  
- Interface responsiva (mobile e desktop)  
- Persistência de dados no navegador (localStorage)  
- Alternância de tema (dark/light mode)  
- Feedback visual com notificações (toast)

##  Como rodar localmente

### 1. Clone o repositório


git clone https://github.com/murilotecoteco/Gerenciador-de-Tarefas.git

### 2. Entre na pasta do arquivo

cd Gerenciador-de-Tarefas

### 3. Instale as dependências

 npm install

### 4. Rode o projeto
 npm run dev

### Build de Produção
 npm run build
 npm start
